# Node Voting Application

NodeJS + MongoDB → Pusher → Single Page Application

## To use

Clone the repo: `git clone https://github.com/bayoishola20/votingApp.git`

`$ (sudo) npm install`

`$ nodemon`

## You should have installed (most recent versions):

* Node
* Express
* MongoDB
* Pusher API
* VanillaJS
